Galley — EPUB Editor & Maker

Getting Started

  • Create a blank EPUB from Library → New, or import EPUB 2/3 via Files / Share Sheet / Open In
  • Edit chapters in WYSIWYG mode; Pro unlocks XHTML/CSS code view and iPad split preview
  • Add, rename, reorder, or delete chapters — spine and navigation stay consistent
  • Use Metadata, Images, TOC, and Validation sheets from the editor
  • Export EPUB 3 via the system Share Sheet (Free). Pro unlocks EPUB 2 export and full validation + auto-fix

Galley Pro

  • One-time Non-Consumable unlock ($9.99): unlimited projects, code view, iPad split editor, TOC tools, image insert/replace/resize, full validation + auto-fix, EPUB 2 export, and optional iCloud Drive sync
  • Purchase or restore from Settings (or the paywall)
  • Free tier keeps core WYSIWYG editing, a 3-project library, basic metadata/cover, basic validation, and EPUB 3 export
  • Galley Pro+ is not offered for sale in v1.0

iCloud Drive sync (Pro)

  • Optional Pro feature using iCloud Drive (Cloud Documents), container iCloud.me.tungngo.Galley
  • Enable in Settings → Sync projects to iCloud Drive, then Sync All or leave the editor to mirror folders
  • On another device: unlock Pro, enable sync, then Restore from iCloud
  • If copies diverge, Galley shows Keep Local / Keep Cloud — not silent last-write-wins

Known Issues & limits (v1.0)

  • DRM-protected EPUB: Galley refuses Adobe ADEPT, Readium LCP, and FairPlay DRM files.
  • Fixed-layout (FXL): Not a full FXL editor in v1.0.
  • PDF / DOCX / Markdown import or export: Not included in v1.0 (on the roadmap).
  • Apple Pencil markup: Not included in v1.0 (system Scribble in text fields may still work where iOS provides it).
  • Third-party cloud OAuth: No Google Drive / Dropbox / OneDrive / WebDAV integration in v1.0 — use Files and Share Sheet.

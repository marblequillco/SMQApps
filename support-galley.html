Galley — EPUB Editor

Getting Started

  • Import an EPUB via the Files app, Share Sheet, or cloud storage (iCloud Drive, Google Drive, Dropbox, OneDrive, WebDAV)
  • Edit in WYSIWYG mode or open the code view for XHTML/CSS
  • Use the chapter sidebar to reorder, add, or delete chapters (spine updates automatically)
  • Edit the table of contents, metadata, and images from their panels
  • Run validation, then export as EPUB 2, EPUB 3, PDF, or DOCX

Galley Pro

  • One-time unlock for code view, split view (iPad), unlimited projects, full TOC/metadata editing, image tools, all export formats, and iCloud Drive sync
  • Purchase or restore from Settings → Galley Pro
  • Core WYSIWYG editing, 3 projects, basic metadata, EPUB 3 export, and validation stay free

Known Issues

  • DRM-protected EPUB: Galley cannot edit Adobe ADEPT, Readium LCP, or FairPlay DRM files — you'll see a clear "cannot edit DRM" notice.
  • Fixed-layout (FXL): EPUB fixed-layout is view-only in v1.0; full FXL editing is planned for v2.0.
  • iCloud sync: Requires iCloud Drive signed in and Galley Pro unlocked. Sync is file-based via iCloud Drive; resolve conflicts with last-write-wins.
  • AI cleanup: The AI-assisted cleanup features are a post-launch IAP and are not available in v1.0.

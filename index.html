Symposium

Galley

Galley

The EPUB editor iPad deserves

The first native EPUB editor for iOS. Bring Sigil-level power — code view, WYSIWYG editing, TOC management, metadata editing, and validation — to a touch-first interface. Edit, restructure, and export EPUB files on your iPad without a laptop.

  • WYSIWYG + code (XHTML/CSS) editing
  • Split view: code + live preview (iPad)
  • Drag-and-drop chapter reordering
  • TOC editor (NCX + nav.xhtml)
  • Full metadata editor with ISBN lookup
  • Built-in validation with auto-fix
  • EPUB 2/3, PDF, and DOCX export
  • iCloud Drive sync + Apple Pencil markup

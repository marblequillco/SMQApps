Marble — EPUB & PDF Reader

Getting Started

  • Import books via the Files app, share sheet, Google Drive, or Dropbox
  • Add an OPDS catalog (e.g. Standard Ebooks) from the Catalogs tab
  • Discover a Calibre Content Server on your local network via Bonjour, or add the server URL manually
  • Open a book and use the Aa controls for fonts, spacing, margins, and themes

Marble Pro

  • One-time unlock for iCloud sync (progress + annotations), reading stats, annotation export (Markdown/CSV), and advanced themes (Leather, Slate)
  • Purchase or restore from Settings → Marble Pro
  • Core local reading (library, OPDS, Calibre, formats above) stays free

Known Issues

  • Calibre not discovered: Confirm the Content Server is running and your iPhone/iPad is on the same Wi‑Fi. Add the host and port manually if Bonjour is blocked.
  • CBZ opens but CBR does not: ZIP-based CBZ is supported; full RAR CBR support is planned.
  • iCloud sync: Only reading positions and annotations sync — book files stay on each device. Ensure iCloud is signed in and Marble Pro is unlocked.
